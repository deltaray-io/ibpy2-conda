# Conda Package Definitions for IbPy2
[IbPy2](https://pypi.python.org/pypi/IbPy2) is a community driven Python connector to Interactive Broker's Trader Workstation. 
This repository contains the Anaconda Package definitions created for IbPy2.

## Python Versions & Platforms supported:
 * Python 2.7, 3.4, 3.5, 3.6 & 3.7
 * 32 and 64bit Linux, OS X and Windows

## Install
To install this package with conda run:
`conda install -c tibkiss ibpy2=0.8.0`
